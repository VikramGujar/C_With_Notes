Introduction to C

C is a:

  • High level / middle level programming language.
  • Compiler-based programming language.
  • Procedure-oriented programming language (POPs).
  • General-purpose programming language.

What is a Program?

Set of instructions is called a program.

What is Software?

Set of programs is called software. We have two types of software:

  • System software (e.g., OS, device drivers, translators)
  • Application software (e.g., WhatsApp, PhonePe, IRCTC)

What is a Language?

Languages are used to communicate with others. For example, human languages like Telugu, English, Hindi, Marathi are used for human communication. But to communicate with machines, we use programming languages like C, C++, Java, .NET, Python, Go, R, etc.

Computer languages are divided into three types:

  • Machine language: Created with binary code (e.g., 11100010).
  • Low-level / assembly language: Created with English-like shortcuts called mnemonics (e.g., gd mrg, sub, add).
  • High-level language: Created with simple English (e.g., good morning, subject, address).

What is a Translator?

Translators convert source code (in English) to binary code that computers can understand. Types of translators include:

  • Compiler: Converts the total program into binary code at once, leaving error lines.
  • Interpreter: Checks the program line by line.
  • Assembler: Converts low-level programs to binary code. Similar to a compiler.

In C & C++, compilers are used. In Java, .NET, and Python, both compilers and interpreters are used, making them compiler-based interpreted languages.

What is a Programming Paradigm?

Every programming language has a structure with specific rules and regulations, called a programming paradigm.

Before C language, we had monolithic programming paradigms, where the entire application was designed with a single program. This made it difficult to find errors, increased memory usage, and reduced performance.

To address these issues, the procedure-oriented programming structure (POPs) was introduced, where a big program is divided into smaller modules or procedures.

Advantages of POP:

  • Modularity: Dividing a big program into small modules according to the project requirement.
  • Simplicity: Easy to read and understand.
  • Reusability: Write once, use many times.
  • Efficiency: High performance.

Disadvantage of POP:

The data is public, meaning any function can access the data from anywhere.

C is a POP language, while C++ & Python are multi-paradigm languages supporting both POP and OOP.

Why is C a General-Purpose Language?

C can be used to develop various types of software, including:

  • Operating systems (e.g., Windows, Linux, macOS, Android, iOS)
  • Editors (e.g., Notepad, WordPad, MS Word)
  • Translators (e.g., Compiler, Interpreter, Assembler)
  • Commercial applications (e.g., Hotel, College, Supermarket programs)
  • Databases (e.g., Oracle, SQL Server, MongoDB)
  • Device drivers (e.g., Audio, Video, Printer, USB drivers)
  • Embedded applications
  • Mobile and PC games
  • Media players (e.g., VLC, MX Player, Windows Media Player)
  • Antivirus (e.g., Avast, NOD, McAfee)
  • Browsers (e.g., Chrome, Firefox, Edge)
  • Standalone applications
