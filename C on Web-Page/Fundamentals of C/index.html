Introduction to C Language

C is a high level and middle level programming language that is compiler-based and procedure-oriented. It is a general-purpose programming language, suitable for system and application software development.

What is a Program?

A set of instructions is called a program.

What is Software?

A set of programs is called software. Software can be classified into two categories:

  • System Software: Examples include operating systems, device drivers, and translators.
  • Application Software: Examples include apps like WhatsApp, PhonePe, and IRCTC.

What is a Language?

Languages are used to communicate. Human languages like Telugu, English, Hindi, and Marathi are used to communicate with humans, whereas programming languages like C, C++, Java, Python, etc., are used to communicate with machines.

Types of Programming Languages

  • Machine Language: Uses binary code (0, 1). Example: 11100010
  • Low Level/Assembly Language: Uses mnemonics (shortcuts). Example: gd mrg
  • High Level Language: Uses simple English. Example: good morning

Example: Adding Two Numbers in C

#include <stdio.h>
int main() {
    int a = 5, b = 10, sum;
    sum = a + b;
    printf("Sum: %d", sum);
    return 0;
}

What is a Translator?

A translator converts high-level code to binary code (machine language). Types of translators:

  • Compiler: Converts the entire program into binary code at once, ignoring error lines.
  • Interpreter: Converts the program line by line.
  • Assembler: Converts low-level programs into binary code, similar to a compiler.

Programming Paradigms

Every programming language follows a specific programming structure called a paradigm.

Procedure-Oriented Programming (POP)

In POP, a big program is divided into smaller sub-programs, also known as procedures, functions, or modules.

  • Advantages of POP:
    • Modularity: Divides a big program into small modules.
    • Simplicity: Easy to read and understand.
    • Reusability: Write once, use many times.
    • Efficiency: High performance.
  • Disadvantages of POP:
    • Data is public and can be accessed from anywhere, leading to security issues.

Object-Oriented Programming (OOP)

OOP is a programming paradigm based on the concept of objects and classes. It is used in languages like C++, Python, Java, and .NET.

  • Class: A blueprint to create objects.
  • Object: An instance of a class.
  • Inheritance: Mechanism of basing a new class on an existing class.
  • Polymorphism: Ability to process objects differently based on their data type or class.
  • Abstraction: Simplification of complex reality by modeling classes appropriate to the problem.

Why C is a General Purpose Language?

C is used to develop various types of software, including:

  • Operating Systems (e.g., Windows, Linux, macOS)
  • Editors (e.g., Notepad, MS Word)
  • Translators (e.g., Compiler, Interpreter)
  • Commercial Applications (e.g., Hotel management software)
  • Databases (e.g., Oracle, SQL Server)
  • Device Drivers (e.g., Printer drivers)
  • Embedded Applications
  • Games
  • Media Players (e.g., VLC, Windows Media Player)
  • Antivirus Software (e.g., Avast, McAfee)
  • Browsers (e.g., Chrome, Firefox)
  • Standalone Applications

Fundamentals of C

C Character Set

C uses the ASCII character set, which consists of 256 characters. The character set includes alphabets, digits, operators, and separators.

Character ASCII Value
a-z 97-122
A-Z 65-90
0-9 48-57
Space 32

C Tokens

Tokens are the smallest individual elements in a C program. They include keywords, identifiers, constants, strings, and operators.

Data Types

Data types define the type of data that can be stored in a variable. C provides the following data types:

  • Primitive Data Types: int, float, char
  • Derived Data Types: array, pointer, function
  • User-Defined Data Types: structure, union, enum

Example: Integer Data Type

int num = 10; // Declares an integer variable 'num' and initializes it with 10

Operators

Operators are symbols that perform operations on variables and values. C includes several types of operators:

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=
  • Increment/Decrement Operators: ++, --
  • Conditional (Ternary) Operator: ? :

Control Flow Statements

Control flow statements control the flow of execution in a program. They include:

  • If Statement: Executes a block of code if a condition is true.
  • Switch Statement: Allows multi-way branching based on the value of a variable.
  • Loops: for, while, and do-while loops for repetitive execution.

Example: If Statement

int a = 5;
if (a > 0) {
    printf("Positive number");
}

Functions

Functions are blocks of code that perform specific tasks. They can be called from other functions and are essential for modular programming.

Example: Function Declaration and Definition

#include <stdio.h>

void greet() {
    printf("Hello, World!");
}

int main() {
    greet(); // Call the function
    return 0;
}

File Handling

C provides functions to handle files, including opening, reading, writing, and closing files.

Example: File Operations

#include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "w"); // Open file for writing
    if (file == NULL) {
        printf("Error opening file!");
        return 1;
    }
    fprintf(file, "Hello, File!"); // Write to file
    fclose(file); // Close file
    return 0;
}

Conclusion

C is a powerful language that has significantly influenced many other programming languages. Its simplicity and efficiency make it a great choice for system and application development.
